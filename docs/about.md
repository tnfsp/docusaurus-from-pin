# 關於我  
  
歡迎來到春熊の心臟醫學！我是春熊，一位專注於心臟外科和重症醫學的醫生。在這個網站上，我將與您分享我的經驗、知識和心得，以及我在這個令人振奮的領域中所面臨的挑戰。

**包含各種文章、短想法與各種蒐集的酷東西**

**為住院醫師、醫學生提供實用的「醫學知識」瀏覽**
